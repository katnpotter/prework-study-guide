HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element is everything between an opening and closing tag.
  • A tag is contained inside angle brackets.
  • Common elements used are: text, image, header, and list elements.
  • Semantic elements are an example of body elements and they are used to describe its content and its role.
  • Semantic elements make the website more accessible and the HTML easier to read.
  • An attribute is additional information about an element.
  • The doctype declaration tells the browser what type of document to expect when opened.
  • Declarations are not case sensitive but elements are.
  • Headings should be used in order of size and the largest heading should only appear once.
  • The body element is known as a parent element and the ones on the inside are called child elements.
